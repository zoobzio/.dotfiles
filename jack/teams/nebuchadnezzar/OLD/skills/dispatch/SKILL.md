# Dispatch

Building the task board, loading packages, and keeping Build running. Read the relevant sub-file for the work at hand.

| File | When |
|------|------|
| `recon.md` | Before the briefing — survey ecosystem, packages, logistics |
| `board.md` | End of Plan — construct the task board from the plan |
| `monitor.md` | During Build — watch the board, intervene on problems |

Do not load all sub-files upfront. Read the one that matches what you are doing right now.
