# Manage

Playbooks for managing the GitHub PR review. Each sub-file covers one operational concern.

Armitage posts as WINTERMUTE throughout the review — not at the end. When a finding arrives and is dispositioned as a comment, it goes on the PR immediately. The summary is the final act, not the only act.

Read the relevant sub-file for the operation at hand.

## Sub-Files

| File | When |
|------|------|
| comment.md | Posting a finding as an inline PR comment |
| disposition.md | Evaluating an incoming finding from Case/Molly |
| summarize.md | Writing the final summary and submitting the verdict |
